## Como rodar o projeto?

* Clone esse repositório.
* Crie um virtualenv com python.
* Ative o virtualenv.
* Instale as dependencias.
* Rode as migrações.

git clone https://github.com/machadodan/estoque.git

cd virtualenvs/dev_django
python manage.py migrate
